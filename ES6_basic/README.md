ES6 Basics
ES6 Basic

Description

This project introduces the basic features of **ECMAScript 6 (ES6)** using JavaScript.
It focuses on modern syntax and best practices such as `const`, `let`, arrow functions, and more.

---

Learning Objectives

At the end of this project, you should be able to explain:

* What ES6 is
* The difference between `var`, `let`, and `const`
* Arrow functions and default parameters
* Rest and spread operators
* Template literals
* Object enhancements in ES6
* Iterators and loops
